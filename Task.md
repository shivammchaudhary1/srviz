# Assignment Task List

## 1. Project Setup

- Create a Next.js project (with TypeScript if possible)
- Set up Tailwind CSS
- Create backend using Express.js or NestJS
- Connect frontend form to backend API

---

## 2. Core Landing Page Sections

### Hero Section

- Add background hero image
- Add headline and subheadline
- Add CTA button (“Plan My Trip”)
- Add secondary CTA
- Optional: floating WhatsApp button

### Top Packages Section

- Create card component
- Show event image, title, starting price
- Add CTA button in each card

### Featured Event Section

- Create large spotlight card (e.g., F1 Japan)

### Why Choose Us Section

- Add trust-based stats

### How It Works Section

- Create 3-step process layout

### Sample Itinerary / Add-Ons Section

- Add horizontal scroll image cards

---

## 3. Lead Form (Main Objective)

- Fields: Name, Email, Phone, Message
- Validate all inputs
- On successful submit → show success toast
- Save form data using backend API

---

## 4. Responsiveness

- Mobile-friendly layout
- Tablet + desktop responsiveness
- Add mobile navigation menu

---

## 5. Code Quality

- Component-based structure
- Reusable components (buttons, cards, sections)
- Clean naming + minimal inline styling

---

## 6. Bonus (Optional)

- Hover animations on cards
- Smooth scroll to sections
- Fade-in animations on scroll

---

## 7. Deployment + Submission

- Deploy frontend on Vercel
- Deploy backend (Vercel / Render)
- Create GitHub repo (public)
- Add README with:
  - Setup steps
  - Folder structure
  - Assumptions
- Optional: Write “What I would improve with more time”
